# Themes

Pollarys supports built-in themes and custom theme JSON files.

## Add a custom theme

1. Create the themes directory:

```bash
mkdir -p ~/.config/pollarys/themes
```

2. Add your JSON file to that directory (for example `my-theme.json`).
3. Open Pollarys, then go to **Settings -> Theme -> Reload themes**.
4. Pick your theme from the dropdown.

## Theme location

- macOS/Linux: `~/.config/pollarys/themes/`
