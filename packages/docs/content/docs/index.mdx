# Pollarys Docs

Pollarys is the visual workspace around OpenCode.

Use these docs to:

- install the right surface for your workflow
- expose Pollarys safely for remote use
- customize appearance and troubleshoot common issues

## What Pollarys is for

Pollarys is for the parts of AI coding that benefit from a control room: branching sessions, reviewing diffs, managing terminals, watching tool progress, running project actions, and keeping the full board visible while the agent works.
